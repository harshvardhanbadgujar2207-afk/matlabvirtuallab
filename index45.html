Systems of Linear Equations

Introduction, Examples, Quiz, and Creators

1) Introduction

A system of linear equations is a set of equations where each equation is linear in the unknowns. It can be written in matrix form as A·x = b.

  • Consistent: The system has at least one solution.
  • Inconsistent: The system has no solution.
  • Unique Solution: The system has exactly one solution (rank(A) = rank([A|b]) = number of unknowns).
  • Infinite Solutions: The system has infinitely many solutions (rank(A) = rank([A|b]) < number of unknowns).

2) Try Out Some Examples

Unique Solution:
Equations: x + y = 2, 2x - y = 1
Matrix A = [[1,1],[2,-1]], rank(A)=2, rank([A|b])=2 → Unique solution.
Infinite Solutions:
Equations: x + y = 2, 2x + 2y = 4
Matrix A = [[1,1],[2,2]], rank(A)=1, rank([A|b])=1 < 2 → Infinite solutions.
Inconsistent System:
Equations: x + y = 2, x + y = 3
Matrix A = [[1,1],[1,1]], rank(A)=1, rank([A|b])=2 → No solution.

3) Test Your Knowledge

Q1: If rank(A)=rank([A|b])=n, what type of solution does the system have?

Q2: If rank(A) < rank([A|b]), what type of system is it?

Q3: If rank(A)=rank([A|b]) < n, what type of solution does the system have?
